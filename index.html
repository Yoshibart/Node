PROGRAM

The following website merges different types of data such country, city, continent, domain, coastline, flag, currency. This is achieved through using Node to provides resources that are stored using the JSON format. Resources are then accesssed using AJAX which when all data has been read, its render using the table. The website also implements effects using JQuery on buttons, which ehen click they provide functionality such as showing the first 20 rows or all rows of the table, hide or show the flags column, fadeIn or fadeOut the THead, SlideOut or SlideIn the currency column.

TECHNOLOGY USED

Technologies used include:-
JSON(JavaScript Object Notation) is a data structure used for storing and retrieving data. Data is stored using key and value format. JSON is light-weight and is also programming language independent since it follows the same format as dictionary in python, object in JavaScript and others.
AJAX(Asynchronous JavaScript and XML) is a web technology used to provide resources through asynchronous communication, which means resources are provided only when ready. AJAX is mainly used in updating different section of the website without need for reloading the whole website.
NODE is a backend JavaScript technology which is used in the implementation of the server, which provides resources for the frontend.
JQUERY is a JavaScript library which provide functionality for writing JavaScript faster.
